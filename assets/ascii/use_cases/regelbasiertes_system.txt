                           ⬇ Vergleich ⬇


================================================================
|                        KI-System                             |
|--------------------------------------------------------------|
| Beispiel:                                                    |
| Eingabe: "Was ergibt 2 + 2 im Kontext von Zeitreisen?"       |
| Modell: Sprachmodell mit Kontextverarbeitung                 |
| Ausgabe:                                                     |
| - "Mathematisch 4, aber in einem hypothetischen Szenario..." |
| - "Wenn sich die Realität verschiebt, könnten Ergebnisse..." |
|                                                              |
| 📌 Output kann bei gleicher Eingabe variieren                |
| 📎 Abhängig von Trainingsdaten, Kontext, Wahrscheinlichkeiten|
|                                                              |
| ➤ Lernfähig, kreativ, probabilistisch                        |
| ➤ KI-System i.S.v. Art. 3 Nr. 1 EU AI Act                    |
